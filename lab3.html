In this lab, we are focused on creating a frontend that is responsive.

In order to create a responsive frontend, we use Bootstrap. Bootstrap is a CSS framework that makes developing web pages easier. Bootstrap provides multiple coded componentes and easier ways to apply styles to html elements.
